**Objective:**
You must implement the following formula as a function:    
$$
\nabla(J) = \frac{1}{m} x^T(X\theta - y)
$$  

where  
- $\nabla(J)$ is a vector of size n 
- $x$ is a matrix of size m * n 
- $y$ is a vector of size m * 1
- $\theta$ is a vector of size n * 1
